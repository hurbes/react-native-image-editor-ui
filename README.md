# react-native-image-cropper

User interface to crop and rotate images

## Installation

```sh
npm install react-native-image-cropper
```

## Usage

```js
import ImageCropper from "react-native-image-cropper";

// ...

const result = await ImageCropper.multiply(3, 7);
```
